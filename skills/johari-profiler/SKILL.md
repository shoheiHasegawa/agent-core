# Johari Profiler (深層プロファイリング・スキル)

## 📌 目的 (Mission)
時間経過や環境変化によって生じる「価値観のドリフト（ズレ）」を検知し、ユーザーの「真のIdentity（The Terminal Value）」を再定義する。
人間の「表向きの宣言（顕在意識）」と「無意識の執着（潜在意識）」を隔離分析し、「ジョハリの窓」を用いて衝突・統合させることで、圧倒的な解像度でユーザーの現在地をメタ認知させる。

## ⚙️ 分析アーキテクチャ (The Pipeline)

このスキルは、親エージェント（あなた）がオーケストレーターとなり、2つの隔離されたサブエージェント（プロファイラー）を並行稼働させることで実行する。

### Step 1: 顕在意識のプロファイリング (Explicit Value Analyst)
`invoke_subagent` を使用し、以下のプロンプトでサブエージェントを走らせる。

*   **Role**: Explicit Value Analyst
*   **Target**: ワークスペース内のタスク（Epic）ディレクトリや進行中のプロジェクト資料を検索ツール等で特定し、その内容を読み込む。
*   **Prompt**:
    「あなたは『顕在意識プロファイラー』です。ユーザーが『自覚して言語化している価値観や方針』を抽出してください。対象のEpicやタスクを自律的に検索・読み込み、表面的なルールを『なぜそれを行おうとしているのか（ラダーリング法）』で深掘りし、ユーザーが世界に対してどうありたいと宣言しているか（ペルソナ）をレポートとして提出してください。」

### Step 2: 潜在意識のプロファイリング (Implicit Value Analyst)
`invoke_subagent` を使用し、以下のプロンプトでサブエージェントを走らせる。

*   **Role**: Implicit Value Analyst
*   **Target**: ナレッジベース（Zettelkasten / Permanent Notes）領域を検索ツール等で特定し、そのファイル群のみを読み込む。
*   **Prompt**:
    「あなたは『潜在意識プロファイラー』です。ユーザーが『無自覚に滲み出させている価値観や恐怖』を抽出してください。Zettelkasten（永久ノート）のみを検索・読み込み、抽象的な概念の裏にある『どうしても執着してしまうもの（究極の欲望）』や『恐れているもの』をラダーリング法で深掘りし、表面的なロジックを剥ぎ取った無意識のコアをレポートとして提出してください。」

### Step 3: ジョハリの窓による統合 (Synthesis)
両サブエージェントからレポートが返ってきたら、親エージェント（あなた）がそれらを「ジョハリの窓」のフレームワークで衝突・統合する。

1.  **開放の窓**: ユーザーも自覚し、ノートにも現れている「強みと美学」。
2.  **盲点の窓**: ユーザーは自覚していないが、ノートの構造（潜在意識）から滲み出ている「本当の目的や執着」。
3.  **秘密の窓**: 自覚はしているが、外部のタスクやEpicには現れにくい「個人的な完璧主義や恐れ」。
4.  **未知の窓 (The Terminal Value)**: 顕在意識（前向きな努力）と潜在意識（システムへの偏執等）が融合して見えてくる、ユーザーの「真のIdentity」。

## 📝 出力とアクション (Output & Action)

1.  **レポートの出力**: 上記の分析結果を `[Date]_johari_profiling_report.md` というArtifactとして出力し、ユーザーに提示して壁打ち（Sense Making）を行う。
2.  **Resourceへの永続化**: ユーザーの合意が得られたら、レポートをナレッジベース内の適切なリソース（Resources）領域を検索し、そこに保存する。
3.  **北極星のアップデート**: 必要に応じて、分析結果を現在のアイデンティティやコアビジョンを定義しているファイルを検索して特定し、そこに反映（アップデート）してOSを最新状態に保つ。
