# agent-core 物理ディレクトリ構造マップ

- `actions/`: カスタムアクションやスクリプト群
- `agents/`: 各種サブエージェントの設定・定義
- `config/`: システム全体の設定ファイル
- `docs/`: 構成図やアーキテクチャドキュメント
  - `architecture/`: You_Inc 全体エコシステムの構成図
- `epics/`: エピック（大規模なプロジェクトやゴール）の管理
- `queue/`: 実行待ちのタスクや一時データ（harvest_reviews等）
- `scripts/`: ユーティリティスクリプト
- `skills/`: エージェントの拡張スキル定義
- `templates/`: 各種テンプレート
- `workflows/`: 自動化ワークフローの定義
- `workspaces/`: 各プロジェクトやエピックの作業スペース
