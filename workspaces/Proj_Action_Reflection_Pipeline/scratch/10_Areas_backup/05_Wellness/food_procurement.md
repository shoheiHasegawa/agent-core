# 食材調達規格 (Ingredient Procurement Standards)

**目的**: 自宅に運び込まれる食材のクオリティを一定以上に保ち、無意識に炎症を抑制する環境を作る。

## 🛡️ 脂質（油）の選定規格
細胞膜の質を決定し、全身の炎症レベルを左右する最重要項目。

### 必須条件
- [ ] すべての油は **「圧搾製法（コールドプレス）」** かつ **「遮光瓶」** であること。
- [ ] 化学溶剤（ヘキサン）抽出の油は、体内の細胞劣化を招くため一切の持ち込みを禁止する。

### 推奨・固定リスト
| 分類 | 用途 | 推奨油 | 備考 |
| :--- | :--- | :--- | :--- |
| **加熱用** | 炒め、焼き | 米油 | 酸化耐性が高い。圧搾一番搾り厳守。 |
| **生食用** | 仕上げ、非加熱 | アマニ油、エゴマ油 | オメガ3補給用。加熱厳禁。 |
| **汎用** | ドレッシング等 | EVオリーブオイル | 遮光瓶、低温圧搾、信頼できるブランドのみ。 |

## 🟢 スタメン食材（OKリスト）
腸内環境の多様性を高め、細胞の材料となる食材を固定で調達する。

- **主食**: 
  - もち麦・押し麦（白米に3割〜混ぜる）
  - オートミール（食物繊維、ビタミンB群）
  - 十割そば（ルチン、低GI）
- **発酵食品**: 
  - 生味噌（「生」記載、空気穴あり）
  - 本格キムチ（「熟成発酵」マークあり）
  - 納豆（タレは使わず、天然塩・醤油・エゴマ油がベスト）
- **野菜・海藻**: 
  - きのこ類全般（常に3種以上ミックス）
  - 根菜類（ごぼう、れんこん、にんじん、さつまいも）
  - ネバネバ系（オクラ、長芋、めかぶ、もずく）
  - アボカド（良質な脂質）
- **タンパク質**: 
  - 豚肉（ビタミンB群、肌のターンオーバー促進）
  - 青魚の水煮缶（サバ、イワシ ※オメガ3豊富）
  - 鶏むね肉、卵、大豆製品

## 🔴 排除食材（NGリスト）
細胞と腸内環境を破壊する「老化促進物質」を生活圏から追放する。

- **狂った油**: サラダ油、キャノーラ油、マーガリン、ショートニング（トランス脂肪酸）
- **超加工食品**: ウインナー、ベーコン、ハム（発色剤等の添加物が腸内細菌を攻撃）
- **老化の元**: 白砂糖、果糖ブドウ糖液糖（ジュース等）、人工甘味料（スクラロース等）
- **過剰な小麦**: 市販の菓子パン、総菜パン
