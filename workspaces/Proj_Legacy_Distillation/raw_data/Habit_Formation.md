# 習慣を作る、生活の仕方
## 習慣化できていること
- 歯医者後のフロスずっと続いている。
## 習慣化したい・できていないこと
###  ランニング習慣化したい
- 全身を鍛えることができる。
- 筋トレも大事だが、動ける体をキープしたい。
- 失われるのは、重いものの持ち上げる能力ではなく、瞬発力
- 止まる動きを鍛えるのが大事、体幹・バランス力を鍛える。
	- 普段鍛えられれない、背面を鍛える。
- やる日を決める？
	- 朝起きて、脳がフレッシュな状態な時間を運動に使う？勉強に使う？
	- 曜日を決めるのがいいか。
### 5時起き
- 何で続かない
	- 睡眠が大事。そもそも睡眠時間が足りていない。
	- 疲れが取れない。目覚めというか。目覚めはいいが、体が元気じゃない
	- 寝る時間が遅い。23時に寝て。5時起きだと6時間
		- 5時起きのための前日の夜の過ごし方ができていない。
	- 外的要因
		- 勉強会リアル参加で、22時に、帰る。
			- そもそもリアル参加の優先順位を高く見積もりすぎてないか。
			- 会社に毒されていないか。自分の軸は？
- 何でしたい、何がしたい
	- 早起きのメリット
		- 5時起きして、9時まで（始業まで）勉強したとすると、朝だけで4時間確保できている。
	- しないことのデメリット
		- 起きる時間が遅くなればなるほど、1日の活動時間が減る。
		- 気づいたら16時とか。
	- どんな状態を目指すか
		- 朝は勉強して、夜は、読書とか生活スタイルを変えたい。
			- やるべき：朝
			- やりたい：夜
