# AWS Certified Data Engineer – Associate（DEA）学習ロードマップ

---

## 🎯 試験の目的と位置づけ
| 項目 | 内容 |
|------|------|
| 対象資格 | AWS Certified Data Engineer – Associate（DEA） |
| 役割 | データの収集・変換・保存・可視化・AI/MLの前処理 |
| あなたの強みとの関連 | DBスペシャリスト＋SQL Silverの知識が強く活きる |
| 学習の意義 | AI開発者としての「データ基盤構築力」を証明 |
| 将来の接続 | Machine Learning Specialty, Data Analytics Specialtyへ直結 |

## 🗂 学習ステップと全体像

| ステップ | 学習内容 | 期間目安 | 使用教材 |
|----------|----------|----------|----------|
| Step 1 | AWS基礎サービスとIAM・VPC概念の整理 | 1〜2週間 | Udemy / AWS公式基礎 |
| Step 2 | データパイプライン構築（S3 / Glue / Kinesis / Lambda） | 3〜4週間 | Udemy / BlackBelt / Hands-on |
| Step 3 | 分析基盤・ETL・ストレージ設計（Redshift / Athena / DynamoDB） | 3週間 | 書籍 / Udemy |
| Step 4 | 模擬試験・本試験対策 | 2週間 | Tutorial Dojo / Whizlabs |
