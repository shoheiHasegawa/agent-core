## ２．シラバスの例「産業教育論～職場における人材育成入門～」
本資料には、シラバスの具体的な記述モデルが収録されています。

### 授業の目的と到達目標
* **目的**: 職場における教育担当者として、効果的、効率的に、深い学びを学習者に提供するために必要となる能力を身につける。
* **到達目標**:
  1. 職場における自己啓発の進め方を説明できる。
  2. 職場におけるOff-JTの進め方を説明できる。
  3. 職場における新人研修プログラムを作ることができる。
  4. 職場におけるOJTの進め方を説明できる。
  5. 職場で使いやすい業務マニュアルを作ることができる。
  6. 職場で求められるコミュニケーション能力を現状よりも向上させる。
  7. モチベーションに関する持（自）論を説明できる。

### 教育方法とスケジュール
* 学習者は「サトウ・フーズワークス株式会社の人事部研修課員」という設定で学ぶシミュレーション法や、マニュアルを作成するプロジェクト法を採用。
* **スケジュールの全体像**:
  * **第1回〜第2回**: オリエンテーション、職場内教育の3類型。
  * **第3回〜第8回**: 第一部「自己啓発」 （自己分析、目標設定、ゲストスピーカ－等）。
  * **第9回〜第16回**: 第二部「Off-JT」 （新人研修プログラム事例分析、コミュニケーション研修体験、プログラム作成・発表）。
  * **第17回〜第20回**: 中間の振り返りと確認テスト、受講生が企画する全体研修。
  * **第21回〜第28回**: 第三部「OJT」 （後輩指導、動機づけ理論、コーチング、叱り方、マニュアル作成手順）。
  * **第29回〜第30回**: 授業のまとめ、最終の振り返り、業務マニュアル作成。

### 成績評価とボーナスポイント（BP）
以下の項目で100ポイント満点中、60ポイント以上を合格とする絶対評価（ポイント加算方式）。
1. 第1部＋第2部確認テスト（20p） / 2. 第3部確認テスト（15p） / 3. 新人研修プログラム一式（10p） / 4. グループワーク相互評価（5p） / 5. マニュアル（10p） / 6. 研修企画・運営評価（5p） / 7. シャトルカードコメント記入（30p）。
* **ボーナスポイント（BP）制度**: 欠席の補填やさらなる能力向上のため、任意の課題（自己分析レポート、組織内教育の提案、優良事例分析、ゲストへの礼状、1分プレゼン等）を提出することでポイントを加算できる。各レポートには詳細な評価ルーブリック（評価観点・尺度・基準）が提示されている。

### 受講ルール
* 授業中の携帯電話・メール・私語・食事は禁止（マナーモードにし、バッグ等にしまう）。
* 欠席は1回につき4ポイント減点。公欠以外の11回以上の欠席で単位不可。グループワークに不参加（睡眠等）の場合も欠席とみなす。

## ３．到達目標に用いられる動詞
* **知識（認知的領域）**: 列挙する、述べる、記述する、説明する、分類する、比較する、予測する、判断する、創造するなど。さらに、「評価」「統合」「分析」「応用」「理解」「知識」の各目標水準に応じた動詞の使い分けが提示されている。
* **技能（精神運動的領域）**: 感じる、模倣する、操作する、測定する、組み立てる、運転するなど。
* **態度・習慣（情意的領域）**: 尋ねる、助ける、討議する、協力する、参加する、配慮するなど。

## ４．学習動機を高めるのに役立つ理論
1. **ARCSモデル**: 注意（Attention）、関連性（Relevance）、自信（Confidence）、満足感（Satisfaction）の4側面から学習意欲をデザインする。
2. **欲求階層説**: マズローによる。低次の欠乏欲求（生理的・安全）から高次の成長欲求（自己実現）への階層。
3. **期待理論**: 「期待（努力で業績が上がる）×道具性（業績が報酬に結びつく）×誘意性（報酬の満足度）」がモチベーションの強さを決定する。
4. **目標設定理論**: 目標の「困難さ」と「明確さ」、フィードバック、コミットメント、自己効力が重要。
5. **公平理論**: 職務に対する労力と報酬のバランスが他者と比較して公平と評価されることが動機づけとなる。
6. **フロー理論**: 関心・集中・喜びを伴い、過剰学習により内発的報酬が最大化される状態。
7. **マインドセット理論**: 固定的知能観（失敗で無力感に陥る）と成長的な知能観（学び自体が目標）の違い。
8. **学習性無力感**: 行動と結果の随伴性がない環境に長く置かれることで生じる「何をやっても無駄」という無力感。
9. **学習動機の2要因モデル**: 市川伸一による分類。「学習内容の重要性」と「学習の功利性」の2軸から、充実志向、訓練志向、実用志向、関係志向、自尊志向、報酬志向の6つの動機と、それに合わせた指導方針を定義。

## ６．研修に役立つ小道具・アプリケーション
| 小道具名 | 活用方法・特徴 |
|---|---|
| **指示棒・レーザーポインタ** | 強調箇所を明示。レーザーポインタなら教室内を動きながら説明可能。 |
| **YES／NOカード** | 多人数講義でも個人に意思表示をさせ、教室全体の傾向を瞬時に確認できる。 |
| **ベル** | グループワーク終了時や私語が気になるときなど、行動や気持ちを切り替える。 |
| **スタンプ** | 「よくできました」等、レポートやカードに短時間で簡単なフィードバックを行う。 |
| **名札・ネームテント** | 教室を記名空間とし、教員や学習者同士のコミュニケーションを円滑にする。 |
| **トランプ・サイコロ** | ランダムな指名、グループ分け、グループ内の役割決定などを遊び心を持って行う。 |
| **模造紙・フリップチャート** | 継続的な情報の掲示や、グループワーク時の書記に。フリップチャートなら可動式で便利。 |
| **書画カメラ** | 平面資料や小さな立体物・実験器具をスクリーンに大きく投影できる。 |
| **クリッカー・アプリ** | クイズやアンケートにスマホで即時回答・集計。匿名性があり参加度が高まる。 |
| **タイマー・アプリ** | スクリーンに残り時間やカウントダウンを表示し、研修の厳密な時間管理を行う。 |

## ７．オンライン研修に関するポイント集
### 研修をオンライン化するための10のポイント
1. 現実的にできることを考える / 2. 学習目標に基づいた研修を設計する / 3. コンテンツは細かく分ける / 4. 学習者を巻き込む / 5. 研修のルールを共有する / 6. すべての学習者が参加できるようにする / 7. 練習をして慣れる / 8. 代替手段を考えておく / 9. 心身ともに健康に気をつける / 10. 完璧であろうと思わない

### オンラインで学習を評価するための10のポイント
1. 教室での一斉試験をオンラインで再現することは難しい / 2. 学問的誠実性について教える / 3. 学習目標を確認する / 4. 形成的評価を積極的に取り入れる / 5. ICTツールを活用して評価する / 6. 問題を工夫する / 7. 解答時間を制限する / 8. 出題パターンを増やす / 9. ピアレビューを導入する / 10. 各種リソースを活用することを推奨する

## ８．ブレンデッド教育とは？
対面とオンラインの学習活動を効果的に組み合わせたカリキュラムおよび研修形態の総称。
* **ローテーション型研修**: 一定時間ごとに学習者が異なる学習形態（対面、オンライン、協働学習等）を移動してまわる。
* **反転研修**: 動画教材等で事前予習し、教室では個別指導や高度な協働学習を行う。
* **分散型研修**: 学習者をグループ分けし、一方を対面で、もう一方をオンラインで実施して交代する。
* **ハイフレックス（HyFlex）型研修**: 対面、同期オンライン、非同期オンラインが同時に提供され、学習者が自在に選択できる。
