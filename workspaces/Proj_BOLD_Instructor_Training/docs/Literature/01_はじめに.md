# １．はじめに

## １－１）研修概要
### ◆ 研修目的
わかりやすく、記憶に残り、効果的な研修を実施し、学習者を学習に参加させるために必要となる基礎的な知識と技術を学びます。具体的には，インストラクショナル・デザインの知見に基づき、研修の構想・設計・実施・評価という一連の過程を体験しながら、求められる能力を身につけます。

### ◆ 学習目標
① 学習ニーズの分析ができるようになる。
② 学習者の学びを促進するシラバスを書けるようになる。
③ 講義法を使って知識をわかりやすく記憶に残るように伝達できるようになる。
④ 学習への参加を取り入れて教えることができるようになる。
⑤ 学習目標に対応する評価方法を選択し、正しく評価することができるようになる。
⑥ 同僚とともに研修改善に取り組むことができるようになる。

## １－２）インストラクショナル・デザイン&スキルとは？

### インストラショナル・デザイン（Instructional Design）
「効率的・効果的な教育を実施するためのシステマティックな手法。ID は、教育プロダクト（授業、カリキュラム、教材）を、ニーズ分析・設計・開発・実施・評価という手順を経て運用するという工学的なアプローチに基づいている。これによって教育者・学習者双方のニーズが満たされ、目標が達成されることを目的としている。第二次世界大戦中のアメリカにおいて新兵教育を短期間で効率よく行うために開発された手法が元になっている。大学がエリート段階の教育機関であった時代にID は個々の教員に私事化されていたが、大衆化段階以降は大量の学習者を効率的・効果的に教育する必要が生じ、かつ ICT 教育が普及するのにあわせて、システム化・標準化を目指す ID の必要度は高めている。ID の専門家であるインストラクショナル・デザイナーには、内容面での専門家と分業・連携して、一連のシステム運用を担うことが期待されている。」
（佐藤浩章（2018）「インストラクショナル・デザイン」『大学を知る辞典』平凡社）

**ADDIE モデルের基本構造**
1. 分析（Analysis）
2. 設計（Design）
3. 開発（Development）
4. 実施（Implementation）
5. 評価（Evaluation）
※これら5つのステップを順番に経て、評価から再度分析へとループするサイクルモデルとなっています。

### インストラクショナル・スキル（Instructional Skills）
ID の中でも、「実施」段階に求められる教授技術。効率的、効果的、魅力的に教える技法。声、表情、ボディランゲージ、姿勢等の身体のコントロール技術、教材・教具の活用技術、発問・指示の技術などがある。学習者の目標達成が成功の指標となる。そのため、インパクトを重視したビジネスプレゼンテーションに求められるスキルなどとは異なる。
